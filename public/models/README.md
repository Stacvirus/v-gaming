# 3D Models

Place your .glb files here:
- arcade-machine.glb  → used by components/3d/ArcadeMachineScene.tsx
- vr-headset.glb      → used by components/3d/VRHeadsetScene.tsx
- billiard-table.glb  → used by components/3d/BilliardScene.tsx

Until models are present, a wireframe icosahedron placeholder renders instead.
Find free CC0 models at: https://sketchfab.com/features/free-3d-models or https://polyhaven.com/models
Use low-poly (<50k triangles) optimized .glb files for best web performance.
